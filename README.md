# Customer Churn Analysis - TelecomX

| Sección                            | Contenido                                                                                                                                                                                                                                                                                                                                                              |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 📌 **Descripción General**         | Este proyecto analiza el comportamiento de cancelación de clientes (churn) en una empresa de telecomunicaciones, con el objetivo de identificar los factores que influyen en la pérdida de clientes y apoyar estrategias de retención.                                                                                                                                 |
| 🎯 **Objetivo de Negocio**         | Identificar patrones y variables clave asociadas al churn para generar insights accionables que permitan reducir la tasa de cancelación.                                                                                                                                                                                                                               |
| 🗂️ **Dataset**                    | Dataset con más de 7000 clientes, incluyendo información demográfica, servicios contratados, tipo de contrato, método de pago, cargos mensuales, antigüedad y estado de churn (0 = activo, 1 = cancelado).                                                                                                                                                             |
| 🧹 **Preparación de Datos**        | - Limpieza de datos y manejo de valores nulos <br> - Transformación de variables categóricas a formato numérico <br> - Conversión de tipos de datos <br> - Creación de variables derivadas <br> - Estandarización de nombres de columnas                                                                                                                               |
| 📈 **Análisis Exploratorio (EDA)** | Se analizaron variables clave mediante visualizaciones para identificar relaciones con el churn: <br><br> - Métodos de pago <br> - Antigüedad del cliente <br> - Cargos mensuales <br> - Servicios contratados <br> - Segmentación de clientes (ej: adultos mayores)                                                                                                   |
| 🔎 **Principales Hallazgos**       | - El método de pago **Electronic Check** presenta mayor tasa de churn <br> - Clientes con **fibra óptica** tienen mayor probabilidad de cancelación <br> - La ausencia de **soporte técnico** aumenta significativamente el churn <br> - La mayoría de cancelaciones ocurre en los **primeros meses** <br> - Mayores cargos mensuales están asociados a mayor abandono |
| 💡 **Recomendaciones de Negocio**  | - Investigar problemas en el método de pago Electronic Check <br> - Analizar causas del churn en clientes con fibra óptica <br> - Fortalecer el soporte técnico como estrategia de retención <br> - Implementar acciones para retener clientes en etapas tempranas                                                                                                     |
| 🛠️ **Tecnologías**                | Python (Pandas, NumPy), Matplotlib, Seaborn, Jupyter Notebook                                                                                                                                                                                                                                                                                                          |
| ⚠️ **Nota**                        | Proyecto desarrollado como parte de formación en análisis de datos, enfocado en la aplicación de técnicas analíticas a un caso de negocio.                                                                                                                                                                                                                             |
